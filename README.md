## 📥 다운로드
다음 명령어를 실행해 모듈을 다운로드하여 실행하세요.
> [Node.js](https://nodejs.org/ko/) 를 사용하므로, 설치가 필수입니다.<br>
```cmd
npm install
```
```cmd
npm run dev
```
실행을 완료하였다면, [localhost:3000](http://localhost:3000) 에서 확인하세요.
